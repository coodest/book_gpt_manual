\textbf{操作流程}：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  \textbf{描述需求}
  Prompt：
  \prompt{``用Mermaid语法绘制佛教从印度经中亚传入中国的流程图，包含主要节点和传播方式''}
\item
  \textbf{获得代码} GPT输出：
  
\begin{gptbox}
\small{
\begin{lstlisting}
graph LR
 A[印度那烂陀寺] -->|法显陆路取经| B[龟兹]

 C --> E[长安大雁塔]
 D --> E
\end{lstlisting}
}
\end{gptbox}

\item
  \textbf{实时渲染}：将代码粘贴至Markdown编辑器或Mermaid Live
  Editor，立即生成矢量图
  \fig[h]{
